# threat_db — Система управления угрозами ИБ

Десктопное приложение на **C++ / Qt6** для учёта и анализа угроз информационной безопасности.  
Соответствует техническому заданию на курсовую работу по дисциплине «Программирование».

## Функции
- Авторизация и регистрация пользователей
- Просмотр списка угроз
- Добавление/удаление угроз
- Просмотр логов (личных и системных — для админа)
- Статистика по уровням риска

## Запуск через Docker (Linux с X11)

```bash
# Разрешить доступ контейнера к X-серверу
xhost +local:root

# Сборка образа
docker build -t threat_db .

# Запуск GUI-приложения
docker run -it --name threat_container -e DISPLAY=$DISPLAY -v /tmp/.X11-unix:/tmp/.X11-unix threat_db
